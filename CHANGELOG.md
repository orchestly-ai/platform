# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/), and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] - 2025-06-15

### Added

- **Visual Workflow Designer** — Drag-and-drop workflow builder with branching, loops, and error handling
- **Agent Registry** — Register and manage agents across frameworks (LangChain, CrewAI, AutoGen, custom)
- **Smart LLM Router** — Multi-provider routing (OpenAI, Anthropic, Google, Groq, and more) with cost/latency/quality optimization
- **A/B Testing** — Experiment with different LLM configurations and measure user satisfaction
- **Prompt Registry** — Version-controlled prompt templates with rendering and analytics
- **RBAC & Multi-Tenancy** — Fine-grained permissions, roles, and organization isolation
- **Cost Tracking** — Per-agent, per-team, and per-workflow spend tracking with budget alerts
- **Human-in-the-Loop** — Approval workflows for sensitive agent actions
- **Integrations Marketplace** — 400+ pre-built connectors (Slack, GitHub, Jira, Salesforce, and more)
- **Real-time Monitoring** — Live metrics, alerts, and agent health dashboards
- **Audit Logs** — Full audit trail for compliance
- **Scheduler** — Cron and interval-based workflow execution
- **RAG, Memory, MCP** — Built-in support for retrieval-augmented generation, agent memory, and Model Context Protocol
- **Webhooks & Realtime** — Event-driven integrations with WebSocket support
- **Docker Compose** — One-command deployment with PostgreSQL, Redis, API, and dashboard
- **CI/CD Pipeline** — GitHub Actions for testing, linting, and Docker image builds
- **Open-Core Licensing** — Apache 2.0 community edition with optional enterprise features
- **1,100+ Tests** — Comprehensive test suite covering API, services, and integrations
